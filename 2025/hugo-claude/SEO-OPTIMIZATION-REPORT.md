### 2. Schema.org структурированные данные ✅

**Что сделано:**
- Добавлена расширенная Schema.org разметка в `/layouts/partials/head.html`
- Реализован граф данных (@graph) с тремя основными типами:
  - **WebSite** - информация о сайте с поддержкой SearchAction
  - **Organization** - данные о сообществе "Тимлид не кодит"
  - **Article** - динамическая разметка для страниц контента

**Структура Organization:**
```json
{
  "@type": "Organization",
  "name": "Тимлид не кодит",
  "alternateName": "teamleads_kz",
  "url": "https://teamleads.kz/",
  "description": "Крупнейшее русскоязычное сообщество...",
  "foundingDate": "2024",
  "areaServed": { "name": "Казахстан и СНГ" },
  "sameAs": ["https://t.me/teamleads_kz"]
}
```

**Преимущества:**
- Расширенные сниппеты в поисковой выдаче
- Rich results в Google Search
- Улучшенная индексация контента
- Связь с Telegram-каналом

### 4. robots.txt и управление индексацией ✅

**Что настроено:**
- Разрешена индексация всего контента

- Разрешен доступ к изображениям для Googlebot-Image

- Добавлен Crawl-delay для агрессивных ботов (Ahrefs, Semrush)
- Указан путь к sitemap.xml

**Sitemap:**
Hugo автоматически генерирует sitemap.xml благодаря настройке:
```toml
enableRobotsTXT = true
[sitemap]
  changefreq = 'weekly'
  priority = 0.5
```
