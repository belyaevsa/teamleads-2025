## üî¢ Data Requirements

### Source Files
- `messages_export.csv`: All message timestamps
- Monthly aggregations: Pre-computed counts per month
- Hourly breakdown: Messages grouped by hour
- Day-of-week stats: Messages grouped by weekday

### Computed Metrics
```sql
-- Total messages
SELECT COUNT(*) FROM messages;

-- Messages per month
SELECT DATE_TRUNC('month', created_at) as month, COUNT(*)
FROM messages
GROUP BY month;

-- Messages per hour
SELECT EXTRACT(hour FROM created_at) as hour, COUNT(*)
FROM messages
GROUP BY hour;

-- Messages per weekday
SELECT EXTRACT(dow FROM created_at) as weekday, COUNT(*)
FROM messages
GROUP BY weekday;

-- Growth rate
SELECT month,
       count,
       LAG(count) OVER (ORDER BY month) as prev_count,
       (count - LAG(count) OVER (ORDER BY month)) / LAG(count) OVER (ORDER BY month) * 100 as growth_pct
FROM monthly_counts;
```
